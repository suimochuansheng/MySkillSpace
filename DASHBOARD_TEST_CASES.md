## 常见问题排查

### 问题1：登录后无法跳转到仪表盘
**排查步骤：**
1. 检查浏览器控制台是否有错误
2. 检查localStorage中是否存储了user信息
3. 检查后端Session是否创建成功
4. 检查前端API调用是否包含credentials: 'include'

### 问题2：退出登录后仍然显示仪表盘
**排查步骤：**
1. 检查localStorage是否被清除
2. 检查后端logout API是否调用成功
3. 检查Session是否被正确清除

### 问题3：模块切换无反应
**排查步骤：**
1. 检查浏览器控制台是否有Vue组件错误
2. 检查moduleConfig映射是否正确
3. 检查组件导入路径是否正确

### 问题4：API调用失败
**排查步骤：**
1. 检查后端服务器是否运行（http://127.0.0.1:8000/）
2. 检查浏览器Network选项卡查看请求详情
3. 检查CORS配置是否正确
4. 检查API端点路径是否正确
