# 安装说明：

# 1. 安装PyTorch（根据CUDA版本选择）
#    
#    方式A - CUDA 11.8（推荐，有NVIDIA GPU）：
#    pip install torch==2.1.2 torchvision==0.16.2 torchaudio==2.1.2 --index-url https://download.pytorch.org/whl/cu118
#    
#    方式B - CPU版本（无GPU或测试环境）：
#    pip install torch==2.1.2 torchvision==0.16.2 torchaudio==2.1.2 --index-url https://download.pytorch.org/whl/cpu
#    
#    方式C - CUDA 12.1（较新GPU）：
#    pip install torch==2.1.2 torchvision==0.16.2 torchaudio==2.1.2 --index-url https://download.pytorch.org/whl/cu121
# 
# 2. 安装其他依赖：
#    pip install -r requirements_py39.txt
# 
# 3. 验证安装：
#    python -c "import django; print('Django:', django.__version__)"
#    python -c "import torch; print('PyTorch:', torch.__version__); print('CUDA可用:', torch.cuda.is_available())"
#    python -c "from transformers import AutoTokenizer; print('Transformers: OK')"
#    python -c "import celery; print('Celery:', celery.__version__)"
# 
# 4. 环境要求：
#    - Python: 3.9.x
#    - 推荐：Python 3.9.13 或更高版本
#    - 操作系统：Windows 10/11, Ubuntu 18.04+, macOS 10.15+
#    - 内存：至少16GB RAM（运行Qwen-7B需要）
#    - 显存：至少12GB VRAM（GPU加速推荐）
# 
# 5. 使用国内镜像加速（可选）：
#    pip install -r requirements_py39.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
