Python基础语法介绍

1. 变量声明

在Python中，无需显式声明变量的数据类型，可以直接赋值给变量名。
python 
# 定义一个整数变量
age = 25

# 定义一个字符串变量
name = "Alice"

# 输出变量内容
print(name)

2. 基本运算符

Python提供了多种算术运算符来进行数值操作。
python 
# 加法
result = 5 + 3
print(result)  # 输出: 8

# 减法
difference = 7 - 4
print(difference)  # 输出: 3

# 乘法
product = 6 * 2
print(product)  # 输出: 12

# 除法
quotient = 9 / 3
print(quotient)  # 输出: 3.0 (结果为浮点数)

# 整除
integer_division = 9 // 3
print(integer_division)  # 输出: 3 (结果为整数)

3. 控制流语句

控制流语句用于根据条件执行不同的代码块。
if 语句
python 
score = 85

if score >= 90:
    print("优秀")
elif score >= 70:
    print("良好")
else:
    print("需要努力")
for 循环
python 
for i in range(5):
    print(i)  # 输出: 0, 1, 2, 3, 4
while 循环
python 
count = 0
while count < 5:
    print(count)  # 输出: 0, 1, 2, 3, 4
    count += 1
